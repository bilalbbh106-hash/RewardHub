# RewardHub Telegram Bot

بوت تيليجرام لإدارة موقع RewardHub

## الأوامر المتاحة

| الأمر | الوظيفة |
|-------|---------|
| `/start` | بدء البوت |
| `/help` | عرض المساعدة |
| `/stats` | إحصائيات الموقع |
| `/users` | قائمة المستخدمين |
| `/withdrawals` | طلبات السحب |
| `/approve` | قبول سحب |
| `/reject` | رفض سحب |
| `/prizes` | إدارة الجوائز |
| `/addprize` | إضافة جائزة |
| `/deleteprize` | حذف جائزة |
| `/tasks` | إدارة المهام |
| `/addtask` | إضافة مهمة |
| `/deletetask` | حذف مهمة |
| `/smartlinks` | إدارة الروابط الذكية |
| `/addlink` | إضافة رابط |
| `/deletelink` | حذف رابط |
| `/notify` | إرسال إشعار |
| `/antifraud` | مكافحة الغش |
| `/togglefraud` | تفعيل/تعطيل مكافحة الغش |
| `/settings` | إعدادات الموقع |
| `/logs` | سجل العمليات |

## التشغيل

```bash
npm install
npm start
